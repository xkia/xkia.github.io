VSCode 快捷键手册 ​

1. 切换语言模式 ​

键位作用生效模式
Ctrl + Shift + J将当前编辑器的语言模式切换为 JSONC（注释支持的 JSON 格式）适用于 编辑器聚焦 的情况

2. 导航到不同编辑器位置 ​

键位作用生效模式
Ctrl + H导航到 左侧编辑器，将焦点移至左边的编辑区域。普通模式 (Vim 模式)
Ctrl + L导航到 右侧编辑器，将焦点移至右边的编辑区域。普通模式 (Vim 模式)
Ctrl + K导航到 上方编辑器，将焦点移至上方的编辑区域。普通模式 (Vim 模式)
Ctrl + J导航到 下方编辑器，将焦点移至下方的编辑区域。普通模式 (Vim 模式)

3. 显示和切换编辑器 ​

键位作用生效模式
Space ,显示所有打开的编辑器，适用于 Vim 正常模式，当编辑器或输入框未聚焦时可用。正常模式 (Vim 模式)
Tab切换到当前编辑器组中的 下一个编辑器。正常模式、可视模式 (Vim 模式)
Shift + Tab切换到当前编辑器组中的 上一个编辑器。正常模式、可视模式 (Vim 模式)

4. 编辑器侧边栏操作 ​

键位作用生效模式
Space E切换侧边栏的 可见性 并聚焦文件资源管理器。适用于 Vim 正常模式，编辑器聚焦时有效。正常模式 (Vim 模式)
Space E (侧边栏聚焦时)切换侧边栏的 可见性 并聚焦活动编辑器组。正常模式 (Vim 模式)

5. 行移动操作 ​

键位作用生效模式
Shift + J将选中的行 下移，在 Vim 可视行模式 下有效。可视行模式 (Vim 模式)
Shift + K将选中的行 上移，在 Vim 可视行模式 下有效。可视行模式 (Vim 模式)

6. 光标悬停信息与代码操作 ​

键位作用生效模式
Shift + K显示光标下符号的 悬停信息（例如函数签名、文档注释等）。正常模式 (Vim 模式)
Space C A触发 代码操作（如修复或重构代码）。正常模式 (Vim 模式)
Space C R重命名光标下的符号。正常模式 (Vim 模式)
Space C S跳转到 代码中的符号（如函数、类等）。正常模式 (Vim 模式)

7. 编辑器和文件操作 ​

键位作用生效模式
Shift + K显示光标下符号的 悬停信息（例如函数签名、文档注释等）。正常模式 (Vim 模式)
Space C A触发 代码操作（如修复或重构代码）。正常模式 (Vim 模式)
Space C R重命名光标下的符号。正常模式 (Vim 模式)
Space C S跳转到 代码中的符号（如函数、类等）。正常模式 (Vim 模式)
Space B D关闭当前活动的编辑器。正常模式 (Vim 模式)
Space Space打开 快速打开对话框，用于快速打开文件、符号或命令。正常模式 (Vim 模式)
Space G D跳转到光标下符号的 定义。正常模式 (Vim 模式)
Space G R跳转到光标下符号的 引用。正常模式 (Vim 模式)
Space G I跳转到光标下符号的 实现。正常模式 (Vim 模式)
Space S G在文件中查找文本。正常模式 (Vim 模式)

8. 源代码管理操作 ​

键位作用生效模式
Space G G聚焦 源代码管理面板，方便查看版本控制相关的操作。正常模式 (Vim 模式)
Ctrl + N将 下一个匹配项 添加到选择中。适用于文本查找中。正常模式、可视模式 (Vim 模式)

文件资源管理器操作 ​

1. 文件操作 ​

键位作用生效模式
R重命名选中的文件。适用于 文件资源管理器聚焦时。文件资源管理器聚焦时
C复制选中的文件。适用于 文件资源管理器聚焦时。文件资源管理器聚焦时
P粘贴复制/剪切的文件。适用于 文件资源管理器聚焦时。文件资源管理器聚焦时
X剪切选中的文件。适用于 文件资源管理器聚焦时。文件资源管理器聚焦时
D删除选中的文件。适用于 文件资源管理器聚焦时。文件资源管理器聚焦时

2. 文件夹和新文件操作 ​

键位作用生效模式
A在资源管理器中创建新文件。适用于 文件资源管理器聚焦时。文件资源管理器聚焦时
Shift + A在资源管理器中创建新文件夹。文件资源管理器聚焦时
S将选中的文件/文件夹在侧边打开。文件资源管理器聚焦时
Shift + S拆分编辑器，打开选中的项，并关闭其他编辑器。文件资源管理器聚焦时

3. 打开文件和折叠文件夹 ​

键位作用生效模式
Enter打开选中的文件并聚焦。适用于文件资源管理器。文件资源管理器聚焦时
Enter (文件夹)展开或折叠文件夹。文件资源管理器聚焦时
